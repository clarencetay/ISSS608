Getting Started

Setting up R packages

To start with, ggstatsplot and tidyverse packages will be installed (if not yet) and launched using library().

packages = c('ggstatsplot','tidyverse')

for(p in packages){
  if(!require(p, character.only = T)){
    install.packages(p)
    }
  library(p, character.only = T)
  }
