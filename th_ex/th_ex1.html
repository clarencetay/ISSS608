A Simple Bar Chart

The code chunk below plots a bar chart by using geom_bar() of ggplot2.

ggplot(data = participants_data,
       aes(x=age, fill= educationLevel)) +
  geom_histogram(bins=20, color='grey30')


ggplot(data = participants_data,
       aes(x=joviality, fill= educationLevel)) +
